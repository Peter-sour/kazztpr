apa saja yang saya pelajari untuk menjadi seorang front end development


HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dasar sebuah halaman web. HTML menggunakan tag untuk mengatur elemen seperti teks, gambar, tautan, dan lainnya agar dapat ditampilkan oleh browser.

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan tata letak elemen-elemen HTML. CSS memungkinkan Anda mengubah warna, ukuran, jarak, dan tata letak halaman agar terlihat lebih menarik dan responsif di berbagai perangkat.

JavaScript adalah bahasa pemrograman yang berjalan di sisi klien (client-side) dan digunakan untuk menambah interaktivitas pada halaman web. Dengan JavaScript, Anda bisa membuat elemen-elemen di halaman web lebih dinamis, seperti animasi, validasi formulir, dan lainnya.


Bootstrap adalah framework CSS yang membantu dalam pengembangan desain web yang responsif. Dengan Bootstrap, Anda dapat dengan cepat membuat tata letak yang konsisten menggunakan komponen dan gaya yang sudah disediakan, seperti tombol, navbar, grid, dan lainnya.

Git adalah sistem kontrol versi yang digunakan untuk melacak perubahan pada kode. Git memungkinkan pengembang untuk berkolaborasi dalam proyek, menyimpan versi kode, dan mengatur berbagai perubahan agar pengembangan lebih terstruktur dan aman.

React JS adalah pustaka JavaScript yang digunakan untuk membangun antarmuka pengguna interaktif. Dengan pendekatan berbasis komponen dan Virtual DOM, React mempermudah pengembangan aplikasi web cepat dan efisien.
